The Concept: Selectors as HTML Tags

Every component has a selector that becomes a custom HTML tag

Component Definition

@Component({{ '{' }}
    selector: 'app-user',
    templateUrl: './user.component.html'
  })
  export class UserComponent {{ '{' }}
  @Input() userName = 'youngTech';
  }

Component Usage in Parent Template

  // In app.component.html:
  <!-- Using the component as a custom tag -->
  <app-user></app-user>
  
  <!-- With additional HTML around it -->
  <section class="user-section">
    <h2>Team Members</h2>
    <app-user></app-user>
    <app-user></app-user>
    <app-user></app-user>
  </section>

üìö Key Takeaways

üè∑Ô∏è

Component Selectors

Every component becomes a custom HTML tag. Use <app-user> just like <div> or <button>.

üß©

Composition Over Monolith

Build small, focused components and combine them. This is like using Lego blocks instead of carving from a single block of wood.

üîÅ

Reusability

Use the same component multiple times. Each instance is independent - change one, others aren't affected.

üì¶

Module Imports

Remember to add components to the imports array. Angular needs to know about them before you can use them.
